# 공정선거지원단 수당 관리 시스템

> 선거관리위원회 공정선거지원단 직원의 월별 수당을 자동 계산·출력하는 단일 파일 웹 애플리케이션입니다.

![버전](https://img.shields.io/badge/version-v3.18-blue)
![환경](https://img.shields.io/badge/환경-Chrome%20%7C%20Edge-green)
![라이선스](https://img.shields.io/badge/license-MIT-lightgrey)

---

## 주요 특징

- **단일 파일** — `salary_manager_v3_18.html` 하나로 실행, 별도 서버·설치 불필요
- **완전 오프라인** — 인트라넷 환경에서도 동작, 인터넷 연결 불필요
- **브라우저 저장** — 모든 데이터는 PC 브라우저(localStorage)에만 저장
- **PDF 출력** — 명세서·담당자 확인표·근무 캘린더 브라우저 인쇄로 PDF 저장
- **엑셀 내보내기** — ExcelJS 기반 다중 시트 엑셀 파일 생성

## 기능 목록

### 수당 계산
| 항목 | 내용 |
|------|------|
| 평일근무수당 | 시간대별 가산 자동 적용 (기본/초과/야간) |
| 주휴수당 | 주차별 소정근로일 충족 여부 자동 판단 |
| 법정유급휴일수당 | 공휴일 근무 시 1.5배 자동 적용 |
| 토요일근무수당 | 주 40시간 기준 이하/초과 구분 자동 적용 |
| 시간외·야간수당 | 출퇴근 시간 입력 시 구간별 자동 분류 |
| 반일연차 | 오전반가/오후반가 — 1일 수당 지급 + 0.5일 차감 |
| 유급병가 | 한도 일수(기본 5일) 이내 유급 처리 |
| 성과수당 | 건별 입력, 일괄 적용 지원 |

### 4대보험 및 소득세
- 국민연금 / 건강보험 / 장기요양 / 고용보험 자동 공제
- 국민연금 개인 납부 대상자 별도 처리
- 간이세액표 기반 근로소득세·지방소득세 자동 계산
- 세액표 직접 업로드 지원 (xlsx)

### 실비 및 사례금
- 관내 외근 실비 자동 계산 (관용 10,000원 / 자차 20,000원 / 일)
- 4시간 이상/미만 기준 자동 분류
- 단속차량사례금(210-01) — 자차등록 직원의 외근_자차 일수 기반
- 비과세 한도(기본 200,000원) 설정 가능

### 근무 관리
- 직원별 월별 근무 캘린더 (출근 유형·출퇴근 시간·메모 입력)
- 근무형태: 외근·관용 / 외근·자차 / 내근 구분
- 공휴일 자동 반영 (2025~2028년 기본 제공, 사용자 추가 가능)
- 일요일·공휴일 수당 배율 표시 (근무 수정 모달)

### 관외출장 관리
- 기간·차량·식사 여부·메모 등록
- 출장 기간 캘린더 자동 고정 (삭제 전 수정 불가)
- 여러 직원 동시 출장 일괄 등록 지원
- 여비(일비·식비·교통비 등)는 별도 정산 — 급여에 미포함

### 출력 및 내보내기
- **직원 확인용 명세서** (PDF, 직원별 1페이지)
- **담당자 확인표** (PDF, 반별 집계·소요예산·한글 금액)
- **월간 근무 캘린더** (PDF)
- **엑셀 파일** (지급내역 / 공제내역 / 목별 상세 3개 시트)
- **전체 데이터 백업·복원** (JSON)

### 편의 기능
- 되돌리기 / 다시실행 (Ctrl+Z / Ctrl+Y, 최대 20단계)
- 근무형태·출장 일괄 처리 (빠른 일괄 메뉴)
- 비밀번호 잠금 (설정에서 변경 가능, 기본 0000)
- 구분(반) 변경 이력 자동 기록 및 명세서 표시
- 주 40h·52h 초과 경고 표시

## 시작하기

### 요구 환경
- Chrome 또는 Edge 최신 버전 (권장)
- 별도 설치·서버 불필요

### 실행 방법

1. `salary_manager_v3_18.html`을 Chrome 또는 Edge로 열기
2. 기본 비밀번호 **0000** 입력
3. 좌측 `[+ 추가]` 버튼으로 직원 등록
4. `설정(⚙)` → 기관명·선거 종류·구분별 일급 입력

### 선거 종류 설정

| 선거 종류 | 회계 시스템 | 절상 방식 |
|-----------|-------------|-----------|
| 지방선거 | 지선경비통합시스템 | 절상 없음 |
| 그 외 / 비선거기간 | dBrain | 10원 단위 절상 |

## 데이터 관리

> 모든 데이터는 **이 PC의 브라우저 localStorage**에 저장됩니다.
> 브라우저 데이터 삭제 시 복구 불가합니다.

- **백업**: `내보내기 > 전체 데이터 백업(JSON)` 정기 실행 권장
- **이전**: 백업 JSON 파일을 새 PC에서 복원
- **초기화**: `설정 > 데이터 관리 > 전체 데이터 초기화` (주의!)

=============================================
 공정선거지원단 수당 관리 시스템 v3.18
=============================================

■ 구성 파일
  salary_manager_v3_18.html  - 메인 프로그램 (단일 파일)
  exceljs.min.js             - 엑셀 내보내기 라이브러리 (오프라인용)
  README.txt                 - 본 파일 (도움말)

  ※ 두 파일은 항상 같은 폴더에 있어야 합니다.

■ 시작하기
  1. salary_manager_v3_18.html 을 크롬 또는 엣지 브라우저로 열기
  2. 기본 비밀번호 0000 입력
  3. 좌측 [+ 추가] 버튼으로 직원 등록 시작
  4. 설정(⚙️)에서 기관명·선거 종류·구분별 일급 등 기본 정보 입력

■ 주요 기능
  · 직원별 월 근무 캘린더 입력
  · 수당 자동 계산 (평일/주휴/시간외/법정유급 등)
  · 4대보험·소득세 자동 공제
  · 선거 종류 선택 (지방선거/대통령선거 등 절상방식 자동 적용)
  · 반일 연차 (오전반가/오후반가) - 1일 수당 지급 + 0.5일 차감
  · 관내 외근 실비 자동 계산 (관용/자차 구분, 4시간 이상 기준 적용)
  · 단속차량사례금(210-01) 자동 계산 (자차등록 직원의 외근_자차 일수 기반)
  · 자차등록 체크박스 (직원 정보 탭) — 개인차량 사례금 지급 대상 설정
  · 관외출장 기간·차량·식사 여부·메모 기록 (여비는 별도 정산)
  · 출장 일괄 추가·삭제 (빠른 일괄 기능)
  · 직원 확인용 명세서 출력 (PDF)
  · 담당자 확인용 정리표 출력 (PDF, 가로 출력 권장)
  · 직원별 월 근무 캘린더 출력 (PDF)
  · 엑셀 내보내기 (지급내역 + 공제내역 + 목별상세)
  · 전체 데이터 백업/복원 (JSON)
  · 되돌리기/다시실행 (Ctrl+Z / Ctrl+Y, 최대 20단계, 세션 내)
  · 공휴일 사용자 관리 (추가·삭제, 설정 메뉴)

■ 인트라넷 환경 안내
  · 외부 인터넷 연결 불필요
  · 모든 데이터는 이 PC의 브라우저(localStorage)에만 저장
  · 다른 PC 이전 시: 내보내기 > 전체 데이터 백업(JSON) 사용

■ 선거 종류 설정
  설정 > 선거 종류에서 진행 중인 선거를 선택합니다.
  · 지방선거: 지선경비통합시스템 사용 → 절상 없음
  · 그 외 선거/비선거기간: dBrain 사용 → 10원 단위 절상

■ 단속차량사례금 (210-01)
  개인 소유 차량을 업무에 사용하는 직원에게 지급하는 사례금입니다.

  · 직원 정보 탭 > [🚗 자차등록] 체크박스를 켠 직원에게만 적용
  · 근무형태가 외근_자차인 날을 자동 집계하여 계산
  · 지역 구분은 설정 > 단속차량사례금에서 전체 공통 적용
    - 구 지역   : 15,000원/일 이내
    - 시·군 지역 : 20,000원/일 이내
  · 비과세 한도(기본 200,000원)도 설정에서 변경 가능
  · 자차등록된 직원은 사이드바 이름 옆에 🚗 배지 표시

■ 관외출장 관리
  근무 현황 탭 캘린더 아래 [🚗 관외출장] 패널에서 등록합니다.
  관외 기준: 사무소 소재지 관할구역(시·군) 외 이동

  · 등록 항목: 시작일·종료일·차량 종류·조식/석식 여부·메모
  · 메모: 출장 목적·방문지 등 간단한 내용 기록 가능
    → 명세서·정리표·캘린더 출력물에 모두 반영됩니다
  · 일비·식비·운임(교통비·주유비)·숙박비는 여비정산신청서로 별도 정산
    → 이 시스템에서는 급여 계산에 포함되지 않습니다
  · 출장 등록 시 해당 기간 캘린더 자동 고정 (삭제 전 수정 불가)
  · 계약 기간 외 날짜로 출장 등록 시 경고 후 진행 여부 확인
  · 여러 직원 동시 출장: 빠른 일괄 > [✈️ 출장 일괄 관리] 사용

  ※ 관내 활동(관할구역 내 순찰 등) 실비는
    설정 > 실비 설정 (관용 10,000원/자차 20,000원/일)에서 관리

■ 엑셀 내보내기 - 목별 상세 시트
  · 관외출장여비 열(노란 배경): 정산 완료 후 금액을 직접 입력
    → 실수령 열이 자동으로 재계산됩니다 (비과세 항목)
  · 원천세 신고금액은 관외출장여비를 포함하지 않습니다 (비과세)

■ 빠른 일괄 메뉴
  · 수당·공제 일괄 입력: 성과수당·정산보험료 전원 일괄 적용
  · 출장 일괄 관리: 같은 출장 일정 전원·선택 직원에게 일괄 추가/삭제
  · 근무일수 초기화: 해당 월 캘린더 전체 초기화 (확인 후 실행)
  · 근무형태 전원 변경: 전 직원 근무형태 일괄 전환

■ 공휴일 관리
  설정 > 공휴일 관리에서 공휴일을 직접 추가하거나 기본 공휴일을 삭제할 수 있습니다.
  기본 공휴일 데이터: 2025년~2028년 (한국천문연구원 기준)

  · 날짜 추가: 원하는 날짜와 이름을 입력하면 캘린더에 공휴일로 반영
  · 기본 공휴일 삭제: 삭제 표시하면 해당 날짜가 평일로 처리됨
  · 사용자 지정 공휴일은 기본 공휴일보다 우선 적용
  · 연도별 초기화: 해당 연도의 사용자 지정 변경 사항만 일괄 되돌리기
  ※ 설정은 이 PC의 브라우저(localStorage)에 저장됩니다.
    전체 데이터 백업(JSON) 시 공휴일 설정도 함께 포함됩니다.

■ 되돌리기 / 다시실행
  · Ctrl+Z : 마지막 작업 되돌리기 (최대 20단계)
  · Ctrl+Y : 되돌린 작업 다시실행
  · 화면 상단 ↩ / ↪ 버튼으로도 동일하게 사용 가능
  · 세션(탭) 단위로 유지되며, 페이지 새로고침 시 초기화됩니다
  · 텍스트 입력 중에는 브라우저 기본 단축키가 우선 적용됩니다

■ 관리자 기능 (설정 내에서 직접 처리)
  · 비밀번호 초기화 : 설정 > 보안 > 비밀번호 초기화 (0000으로 재설정)
  · 비밀번호 변경   : 설정 > 보안 > 비밀번호 변경
  · 단속사례금 지역 : 설정 > 단속차량사례금 > 지역 구분 (구/시·군)
  · 비과세 한도 변경: 설정 > 단속차량사례금 > 비과세 한도
  · 공휴일 관리     : 설정 > 공휴일 관리 > 날짜 추가·삭제·연도 초기화
  · 전체 데이터 초기화: 설정 > 데이터 관리 > 전체 데이터 초기화 (주의!)

■ 지원 환경
  · 권장: 크롬(Chrome) 또는 엣지(Edge) 최신 버전
  · 지원 연도: 2025년 ~ 2028년 (공휴일 데이터 기준)

■ 데이터 백업 권장
  정기적으로 [내보내기 > 전체 데이터 백업(JSON)]을 실행하여
  데이터를 저장해 두세요. 브라우저 데이터 삭제 시 복구 불가합니다.

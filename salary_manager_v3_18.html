직원 추가왼쪽 사이드바 상단의 + 추가 버튼을 누르면 새 직원을 등록할 수 있습니다. 이름·은행·계좌·계약기간·구분(일급 기준)을 입력하세요.
직원 정보 수정사이드바에서 직원을 선택한 뒤 직원 정보 탭에서 수정할 수 있습니다. 수정 내용은 자동으로 저장됩니다.

자차등록 (개인차량 사례금)직원 정보 탭 하단의 자차등록 체크박스를 켜면 해당 직원의 외근_자차 근무일에 단속차량사례금(210-01)이 자동으로 계산됩니다. 체크하지 않으면 사례금이 발생하지 않습니다. 자차등록된 직원은 사이드바 이름 옆에 배지가 표시됩니다.
구분 변경 이력직원 정보 탭에서 + 구분 변경 추가를 누르면 월 중간에 구분(지역반↔본부반 등)이 바뀌는 경우를 처리할 수 있습니다. 변경 날짜·구분·기본 근무형태를 함께 설정하면 해당 날짜 이후부터 새 단가와 근무형태가 자동 적용됩니다. 구분 변경이 있는 달의 명세서에는 혼합단가로 표시되고 하단에 변경 이력 비고가 출력됩니다.
직원 목록은 구분 기준으로 정렬됩니다. 구분은 설정에서 추가·수정할 수 있습니다.
근무 현황 캘린더
날짜 클릭으로 수정캘린더의 날짜 칸을 클릭하면 해당 날의 근무 상태·출퇴근 시간·근무 형태·메모를 입력하는 창이 열립니다.
캘린더 셀 색상 안내초록: 일반근무 / 주황: 초과근무 / 보라: 야간근무 / 노랑: 공휴일 / 파랑: 토요일 / 빨강: 일요일. 날짜 옆 주황 점(●)은 기본값에서 수정된 날입니다. 주간 실근무시간이 40시간을 넘으면 해당 주 셀이 노란 배경(⚠️연장), 52시간을 넘으면 빨간 배경(🚨초과)으로 강조됩니다.
주간 근무시간 경고주 실근무시간이 40시간을 초과하면 사이드바 직원명 옆에 ⚠️, 52시간을 초과하면 🚨가 표시됩니다. 헤더에도 해당 인원 수가 표시됩니다. 주 52시간은 법정 최대 근로시간(기본 40h + 연장 12h)입니다.
출퇴근 시간 입력근무 선택 후 출근·퇴근 시간을 입력하면 기본 수당·초과 수당·야간 수당이 자동으로 계산됩니다. 야간 기준은 22:00~06:00입니다.
근무 형태 선택외근_관용 / 외근_자차 / 내근 중 하나를 선택하면 실비(교통비)가 자동으로 반영됩니다. 외근_자차를 선택한 날은 관내 실비(자차)도 계산되며, 직원 정보에서 자차등록이 체크된 경우 단속차량사례금(210-01)도 해당 일수만큼 자동 집계됩니다.
반일 연차 (오전/오후)캘린더에서 오전반가 또는 오후반가를 선택하면 해당 날 1일 수당이 전액 지급되고 연차 0.5일이 차감됩니다. 월차 현황에서 이번 달 사용 일수에 0.5일로 반영됩니다.
관외출장 관리근무 현황 탭 캘린더 아래 관외출장 패널에서 출장을 추가합니다. 시작일·종료일·차량 종류·조식/석식 여부·메모(출장 목적·장소 등)를 입력하며, 일비·식비·운임·숙박 전액은 별도 정산합니다 (급여 계산에 포함되지 않습니다). 출장 등록 시 해당 기간의 캘린더가 자동으로 근무 상태로 고정되며, 삭제 전에는 캘린더에서 수정할 수 없습니다. 여러 직원이 함께 출장하는 경우 빠른 일괄 → 출장 일괄 관리를 이용하면 한 번에 추가·삭제할 수 있으며, 메모도 일괄 적용됩니다.
근무시간대 변경 (페인트 모드)캘린더 상단 근무시간대 변경 버튼으로 변경 모드에 진입합니다. 시간대를 선택한 뒤 원하는 날짜를 클릭하면 해당 날의 출퇴근 시간이 일괄 적용됩니다. 완료 후 확인을 눌러 저장하세요.
이번 달 초기화캘린더 상단 이번 달 초기화 버튼을 누르면 현재 선택된 직원의 해당 월 근무 데이터만 초기 상태로 되돌립니다. 다른 직원이나 다른 달에는 영향을 주지 않습니다.
기본값으로 초기화날짜 수정 창 하단의 기본값으로 버튼을 누르면 해당 날짜를 계약 기간·요일 기준 초기 상태로 되돌립니다.
일괄 수정 / 빠른 일괄헤더의 수당·공제 일괄 버튼으로 성과수당·정산보험료를 전체 직원에게 한 번에 입력할 수 있습니다. 왼쪽 사이드바 하단 빠른 일괄 패널에서는 ▸ 근무일수 초기화(확인 팝업 후 실행) ▸ 근무형태 전원 변경 ▸ 출장 일괄 추가·삭제를 빠르게 처리할 수 있습니다. 출장 일괄 관리는 직원 카드를 선택/해제해 대상을 지정하고, 추가 시 기간·차량·식사·메모를 함께 입력합니다.
되돌리기 / 다시실행헤더의 ↩ 되돌리기 버튼(또는 Ctrl+Z)으로 직전 작업을 최대 20단계까지 되돌릴 수 있습니다. ↪ 다시실행(또는 Ctrl+Y)으로 취소한 작업을 다시 적용합니다. 추적 대상은 날짜 근무 수정, 근무일수 초기화, 근무형태 일괄 변경, 수당·공제 일괄 입력, 출장 추가·수정·삭제, 직원 삭제입니다. 단, 페이지 새로고침 시 기록은 초기화됩니다.

비밀번호 변경설정 → 보안 → 비밀번호 변경에서 현재 비밀번호 확인 후 새 비밀번호로 변경할 수 있습니다.
비밀번호 초기화비밀번호를 잊어버린 경우 설정 → 보안 → 비밀번호 초기화로 0000으로 되돌릴 수 있습니다.
우클릭, F12, Ctrl+U 등 개발자 도구 접근은 보안상 차단되어 있습니다.
데이터 저장 방식
브라우저 로컬 저장모든 데이터는 이 컴퓨터의 브라우저(로컬 스토리지)에 저장됩니다. 인터넷이 없어도 동작하며, 서버로 전송되지 않습니다.
주의사항브라우저 캐시를 지우거나 다른 브라우저를 사용하면 데이터가 보이지 않을 수 있습니다. 중요한 데이터는 정기적으로 전체 데이터 백업(JSON)을 저장해 두세요.
관리자 기능설정 → 보안 메뉴에서 비밀번호 초기화(0000으로 리셋)를 할 수 있습니다. 설정 → 데이터 관리에서 전체 데이터 초기화가 가능합니다. 단속차량사례금 지역 구분·비과세 한도는 설정 → 단속차량사례금에서, 공휴일 추가·삭제는 설정 → 공휴일 관리에서 처리합니다.
파일 구성이 프로그램은 salary_manager_v3_18.html과 exceljs.min.js 두 파일로 동작합니다. 두 파일은 항상 같은 폴더에 있어야 합니다.
비밀번호 초기화

비밀번호를 기본값 0000으로 초기화합니다.
수당 데이터는 영향 없습니다.

· PIN이 기본값(0000)으로 리셋됩니다
· localStorage의 수당 데이터는 유지됩니다
· 초기화 후 반드시 새 비밀번호로 변경하세요
